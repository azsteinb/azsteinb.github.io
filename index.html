Bio

I am a software engineer who loves systems programming and web development. I currently work as a Software Engineer Consultant at OrganizeMyPeople, a software firm, where I create full stack web applications.

I have experience with socket programming, encryption, compression, encoding, machine learning, databases, and web development in a variety of frameworks, languages, and technologies.

I love creating my own projects, and I have some of them on my Github. I also have some featured on this website, listed below.

Projects:
Bflock
Bflock, or "Better File Locks", is a C library that makes concurrent file operations easy and safe. Bflock started out as a group of helper functions in a load balancer, but it now stands on its own.
Medibill
Medibill is a tool that uses large datasets to analyze and help negotiate your medical bills. Medibill was created for CruzHacks 2022, and the project won multiple first place awards.
Locally Imagined
Locally Imagined is an art trading platform designed to help small artists and galleries navigate the cut-throat industry of the art trade.
A Brief Survey of Data Placement in a Geo-Distributed Storage System Using ML
This paper, written for a graduate distributed systems class, surveys machine learning based data placement on geo-distributed systems.
