Bio

I am a software engineer with focuses in security infrastructure and distributed systems. I also work on embedded systems. I am part of the Security Software Engineering team. Our team creates systems that adhere to and implement a zero trust architecture.

I have experience with lower level systems programming, higher level system design, and lots of topics in between. Much of my professional experience consists of designing and creating cloud security infrastructure -- and engineering these projects in a way that makes them accessible and "painless" as possible to all involved parties. Recently, I have transitioned from the world of cloud to the world of embedded systems.

I love creating my own projects, and I have some of them on my Github. I also have some featured on this website, listed below. I also from time to time contribute to open source projects.

Projects:
Bflock
Bflock, or "Better File Locks", is a C library that makes concurrent file operations easy and safe. Bflock started out as a group of helper functions in a load balancer, but it now stands on its own.
Medibill
Medibill is a tool that uses large datasets to analyze and help negotiate your medical bills. Medibill was created for CruzHacks 2022, and the project won multiple first place awards.
Locally Imagined
Locally Imagined is an art trading platform designed to help small artists and galleries navigate the cut-throat industry of the art trade.
A Brief Survey of Data Placement in a Geo-Distributed Storage System Using ML
This paper, written for a graduate distributed systems class, surveys machine learning based data placement on geo-distributed systems.
